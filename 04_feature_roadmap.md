# StavidBot Feature Roadmap

Generated by planning agent. Use this as a reference when prioritizing new features.

---

## Phase 1: MVP Playoff Week *(build this first)*
1. Daily Check-in & Pillar Logging
2. Win/Loss Series Tracking
3. Anti-Quit Messaging
4. Playoff Week Dashboard (`/playoff_status`)
5. User Preferences (check-in times, pillar names)

## Phase 2: Core Productivity Integrations
1. Step Count Integration (auto-verify David's 10k step pillar)
2. TikTok Usage Tracking (auto-verify Stephanie's 90min cap)
3. Project Time Tracking (verify David's 30min personal project pillar)
4. Sunday Weekly Review prompt

## Phase 3: Household Management
1. Chore Assignment System (`/chore`, `/chores`, `/chore_complete`)
2. Shared Task Queue (`/task`, `/tasks`, `/task_complete`)
3. Grocery List (`/grocery`)
4. Ledger category tagging + monthly budget review

## Phase 4: Activities & Life
1. Date Night Planning & History
2. Restaurant Finder (Google Places API)
3. Shared Notes / Whiteboard
4. Shared Calendar & Important Dates
5. Achievement Badges / Milestones

## Phase 5: Polish & Advanced
- Leaderboard / head-to-head stats
- Activity Finder (Eventbrite/Ticketmaster)
- Reading Tracker
- Bet/Wager system
- Weekly Themed Challenges
- Data Export (CSV/JSON)
- Subscription & Recurring Payment tracker
- Savings Goal Tracker

## Feature Details

### Habit Tracking (Playoff Week)

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Daily check-in pings + pillar logging | HIGH | `playoff_checkins` table; editable 24h window |
| Win/loss + series tracking | HIGH | `playoff_series`, `playoff_daily_records` tables; auto-reset weekly |
| Anti-quit messaging | HIGH | Show "need X straight" when losing 1-3 or 0-3 |
| Sunday weekly review | HIGH | Scheduled task Sunday 10am ET; modal form |
| `/playoff_status` dashboard | HIGH | Rich embed; color-coded; show next check-in time |
| User preferences (times, pillar names) | HIGH | `user_preferences` table |

### Apartment & Household

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Chore assignment + rotation | HIGH | `chores`, `chore_templates` tables; 24h reminder |
| Ad-hoc task queue | MEDIUM | `tasks` table; optional assignee |
| Grocery list (shared) | MEDIUM | `grocery_items` with check-off; category support |
| Inventory / supplies tracker | LOW | Alert when below threshold |

### Activities & Dating

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Date night history + planning | MEDIUM | `dates` table; track activity types; Saturday tradition |
| Restaurant finder | MEDIUM | Google Places API; store favorites |
| Activity finder | LOW | Eventbrite/Ticketmaster; NYC area |
| Movie/Show/Book watchlist | LOW | TMDB or OMDb API for metadata |

### Communication & Notes

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Shared notes/whiteboard | MEDIUM | `shared_notes` table; for apartment info |
| Shared calendar / important dates | MEDIUM | Weekly Monday summary; birthday/anniversary reminders |
| Ledger category tagging | MEDIUM | Extend `LedgerEntry`; monthly budget review |
| Quote board | LOW | `quotes` table; upvote/random |

### Health & Fitness

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Step count integration | HIGH | Apple Health / Google Fit OAuth; manual fallback |
| Workout logging | MEDIUM | `workouts` table; ties to Stephanie's movement pillar |
| TikTok usage tracking | HIGH | Screen Time API; alert at 60/75min |
| Meal prep / nutrition | LOW | Optional; ties to grocery list |
| Reading tracker | LOW | Ties to Stephanie's finite project pillar |

### Productivity (David's Pillar 3)

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Project time tracking (`/project start/log`) | HIGH | `projects`, `project_sessions` tables |

### Financial

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Utility bill tracking | MEDIUM | Extend ledger; monthly reminders |
| Subscription tracker | MEDIUM | `subscriptions` table; 3-day renewal reminder |

### Gamification

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Achievement badges | MEDIUM | Streak, milestone, cumulative types |
| Leaderboard (head-to-head) | MEDIUM | Aggregate query; weekly or on-demand |
| Wager system | LOW | Fictional points; consider social dynamics |
| Weekly themed challenges | LOW | Opt-in bonus challenges |

### Admin / Bot Health

| Feature | Priority | Key Notes |
|---------|----------|-----------|
| Data export (CSV/JSON) | MEDIUM | Send as Discord file attachment |
| Audit log / edit history | MEDIUM | `updated_at`, `updated_by` on key tables |
| `/status` health check | LOW | Ping DB + external APIs |

## Key New DB Tables Needed

```sql
-- Playoff Week
playoff_series (id, user_id, guild_id, series_start, wins, losses, status)
playoff_daily_records (id, user_id, guild_id, date, won, streak_count)
playoff_checkins (id, user_id, guild_id, date, pillar1, pillar2, pillar3, created_at)
weekly_reviews (id, user_id, guild_id, week_of, review_text, series_result)

-- Household
chores (id, guild_id, name, assigned_to, due_date, completed_at)
chore_templates (name, frequency, description)
tasks (id, guild_id, created_by, assigned_to, title, priority, status, due_date)
grocery_items (id, guild_id, item, quantity, checked_by, checked_at)

-- Tracking
project_sessions (id, guild_id, user_id, project_id, duration_minutes, date)
projects (id, guild_id, user_id, name, status, created_at)
screen_time_logs (id, guild_id, user_id, app, minutes, date)
workouts (id, guild_id, user_id, type, duration_minutes, date)

-- Preferences & Social
user_preferences (user_id, guild_id, check_in_time, timezone, pillar_names)
achievements (id, user_id, guild_id, badge_name, unlocked_at)
```
